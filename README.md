# Institute of Analytics Data Sprint

## Task and Actions
- 4-week data analysis sprint with the Institute of Analytics to clean, analyse and present findings from a music streaming platform dataset.
-	Cleaned data using BigQuery SQL.
-	Financial analysis and revenue projection using Python.
-	Trend analysis in Excel to uncover insights/ patterns in data.

## Results
- Awarded as one of the top 3 contestants. 
-	Met the short project deadline.
-	Enthusiasm to participate in learning and growth opportunities outside of course.
-	Problem-solving ability to clean and analyse datasets to find insights to present to IOA.
